Key highlights of the appAssign Jira issues to GitHub Copilot coding agent, get draft pull requests

More details

GitHub Copilot for Jira can help teams reduce cycle times and increase throughput by automating coding tasks: 

  • Assign GitHub Copilot coding agent via the assignee field 
  • Trigger the agent by mentioning it in issue comments 
  • See updates as pull requests are generated 
  • The agent fetches relevant context from the Jira issues 
  • Use GitHub Copilot for Jira (GHEC with Data Residency) for GitHub Data Residency customers

Requirements: 

  • Jira Cloud instance with Rovo 
  • GitHub users with Copilot coding agent enabled 
  • Connected GitHub repository 

 Use of this integration signals agreement to GitHub's pre-release terms and an instruction to GitHub to send your organization's data to Atlassian. 

Get started: 

  1. Install the apps from the Atlassian and GitHub Marketplaces 
  2. Connect your GitHub organization 
  3. Configure which repositories the agent can access 
  4. Start assigning issues to GitHub Copilot

Summary
GitHub Copilot for Jira: Public Preview
Details

GitHub Copilot coding agent for Jira is now available in public preview. Assign Jira issues to GitHub Copilot coding agent, which will analyze your issue details and open a draft pull request in your GitHub repository—ready for your review.

Features:

  • Copilot analyzes Jira issue descriptions and comments to gather context
  • Works independently to implement changes and open a draft pull request
  • Posts progress updates and asks clarifying questions directly in Jira
  • Keeps your existing review and approval rules for every pull request
