# Knowledge Proposals

这里存放待审核的知识提案，不直接算正式记忆。

推荐结构：

```text
data/knowledge-proposals/YYYY-MM-DD/
├── proposal-001.json
└── proposal-002.json
```

适用场景：

- 需要人审或 captain 审核的长期知识候选
- 低置信度但值得跟踪的知识线索
- 可能影响规则、技能、自动化、长期热缓存的提案

低风险、高置信度、证据明确的内容可由 `aic-curator` 直接落盘，不必强制经过这里。

可直接复制 `TEMPLATE.json` 作为提案骨架。
