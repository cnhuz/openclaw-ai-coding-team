# BOOT.md - Workspace Boot Checklist

启动后先做轻量检查：

1. 读取今天与昨天的日志
2. 检查 `MEMORY.md` 是否有过时条目
3. 检查是否存在卡住的活跃任务
4. 检查前一轮是否留下待同步记忆、待升级事实、待处理 blocker
5. 检查 Git 备份与定时任务是否明显失效；若失效，记录待办并提醒负责人
6. 检查任务真相源与记忆是否出现明显漂移
7. 若是发布相关 Agent，检查环境是否可达
8. 若发现 cron、记忆同步、周级 gate 或 `MEMORY.md` 并发写入异常，先查看 `scripts/README.md`
9. 无需处理时保持安静，不做多余动作
